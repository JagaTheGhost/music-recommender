# Sound Prism: Music AI Recommender

Sound Prism is a hyper-aesthetic, AI-powered music recommendation engine that converts your vibes into DNA-matched Spotify playlists.

![Sound Prism Preview](static/img/preview.png) *(Note: Add a preview screenshot later)*

## ✨ Features

- **🎯 Vibe-to-DNA NLP**: Describe a mood (e.g., "hyper and distorted music for a party") and our pure-Python NLP engine extracts the exact audio features (Energy, Valence, BPM, etc.) to find the perfect tracks.
- **🎨 Sleek Minimalist UI**: A premium dark-mode interface built with glassmorphism, fluid aurora backgrounds, and animated Flaticon icons.
- **🔄 Vibe Sync**: Dynamic background animations that pulse and change speed based on the energy and tempo of the currently playing track.
- **🎲 Roulette Mode**: Feeling indecisive? Use the "Surprise Me" button to generate chaotic, AI-curated vibestrings for instant discovery.
- **🔗 Alchemist Bridge**: Create smooth transitions between songs in your playlist using recursive sonic-midpoint calculation.
- **🛡️ Explicit Filter**: Toggle explicit content on or off with a single click.
- **📱 Fully Responsive**: Optimized for seamless discovery across Desktop and Mobile devices.

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Flask
- Pandas, Numpy, Scikit-learn
- A modern web browser

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd "Music recommender"
   ```
2. Install dependencies:
   ```bash
   pip install flask pandas numpy scikit-learn
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Open your browser and navigate to `http://127.0.0.1:5000`.

## 🛠️ Tech Stack
- **Backend**: Python, Flask
- **Engine**: Scikit-Learn (Cosine Similarity), Custom Regex NLP
- **Frontend**: HTML5, Vanilla CSS3 (Modern Glassmorphism), JavaScript (ES6+)
- **Icons**: Flaticon Uicons
- **Music**: Spotify Embed API & oEmbed Integration
